# FLATDANGO MOVIE TICKET APPLICATION
## Overview
Flatdango is an application for purchasing movie tickets.Users can  view daetils about movies ,check ticket availability and purchase tickets.
## Features
- View available movies 
- Purchase tickets 

## Technologies Used
- HTML
- CSS
- JavaScript
- Json-server

## Set up installation
1. clone the repository
``` bash
git clone git@github.com:happiness-prog/wk3-codechallenge-.git
```
2. Navigate to the project directory
``` bash 
cd wk3-codechallenge-
```
3. Install the required packages
``` bash 
json-server 
```
4. Open index.html
